# How to Run Movies - Complete Guide

### Run Single Movie

python src/run_pipeline.py --movie movie_name
```

### Run All Movies at Once
```powershell
python run_all_movies.py
```

### Run All Movies (Regenerate Fresh)
```powershell
python run_all_movies.py --force
```

### Extract Character Introductions
```powershell
python check_characters.py
```

**outputs**

Check `output_json/` folder for your scene results!
Check `movie_event_json/` folder for character introduction data!
